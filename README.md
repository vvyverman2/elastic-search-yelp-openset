# Description
- Script to upload yelp open dataset to elsatic cloud project
- - Flask app for quering yelp elasticsearch cloud project

## Local Run
### Upload Data
Download yelp open data set https://business.yelp.com/data/resources/open-dataset/ <br />
Add json files into data-injest/data

## Secrets
Add local .env file for secrets from elastic cloud <br />

- API_KEY
